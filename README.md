# Betting Slip Analysis & NBA Prediction Platform

A data-driven decision-support system for sports betting that analyzes betting slips, evaluates risk and value, and generates statistically-justified recommendations.

## Overview

This platform provides intelligent analysis and recommendations for sports betting, starting with NBA basketball. The system focuses on:

- **Value Identification** - Finding bets with positive expected value
- **Risk Assessment** - Evaluating correlation and volatility
- **Contextual Intelligence** - Incorporating injuries, rest, travel, and other factors
- **Transparency** - Explaining every recommendation with statistical justification
